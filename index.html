RSGO Patcher

ReelSteady Go's "Lock Horizon" feature doesn't function properly with the Umma85 and similar setups where the mainboard is mounted offset from the original orientation. Therefore I have created this tool, which allows you to patch your footage before feeding it into RSGO.
